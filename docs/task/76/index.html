Математикалық Метеориттер ← Сабаққа оралу
"Математикалық Метеориттер" — бұл натурал сандардың бөлінгіштік белгілерін (2, 3, 5, 9, 10) бекітуге арналған интерактивті ойын. Тапсырманың мазмұны: Оқушы ғарыш кеңістігінде ұшып жүрген сандар ағынына тап болады. Экранда ағымдағы ереже көрсетіледі (мысалы, "3-ке бөлінетін сандар"). Оқушының міндеті — осы ережеге сәйкес келетін сандарды ғана таңдау. Орындау тәсілі: Ойыншы дұрыс сандарды тінтуірмен басуы немесе сенсорлы экранда саусақпен сырғыту (свайп) арқылы жинауы керек. Ережелер: Дұрыс жауап ұпай қосады. Қате жауап "өмірді" азайтады. Әр 50 ұпай сайын деңгей көтеріліп, бөлінгіштік ережесі өзгереді. 3 пен 9-ға бөлінгіштікті тексеру қиынырақ болғандықтан, ол деңгейлерде сандардың түсу жылдамдығы бәсеңдейді. Бұл тапсырма оқушылардың математикалық логикасын және шешім қабылдау жылдамдығын дамытады.
